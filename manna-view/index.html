Manna-View

Public map intelligence for lawful public-world visibility.

Manna-View is a place-first public map-intelligence project centered on lawful public signals, routes, oversight layers, and compressed place intelligence. It is being built to make public-world visibility cleaner, faster, and more spatially understandable.

See the wider project network Back to manna-core.dev
Mission

Search a place, gather lawful public signals, and map them cleanly for awareness, oversight, and context.

Scope

Cameras, routes, alerts, weather, aircraft, vessels, and other public map-visible data layers.

Boundary

Built for public transparency and public-interest intelligence, not tactical surveillance or evasion.

What It Does

Clean map-first public intelligence

Core Behavior
  • Keeps the map clean with grouping, filters, and progressive reveal
  • Supports place-first analysis instead of clutter-first browsing
  • Compresses local context into readable public intelligence
Why It Exists

The goal is to create a map-first public intelligence workspace that helps people understand the visible public world through geography, movement, and context.

Current Direction

Where it sits inside the network

Map-First

The intended home is a fullscreen map surface with minimizable tools and popup-first workflow.

Public Signals

The project focuses on lawful public data and explicit source visibility.
