Released Utility / Snips

Public releaseWindowsClipboard firstNo telemetry

A screenshot tool where copy really means temporary.

Manna Snips is a small Windows app for taking a snip, marking it up if needed, copying it, and pasting it where it belongs.

The core promise is simple: Copy stays clipboard-first and temporary. Download is the explicit durable-file action.

Current reality

Manna Snips has a public release baseline, installer lane, local settings, hotkey presets, editor tools, and a written privacy story.

Good fit

Bug reports, support chats, docs, tickets, quick notes, and anywhere a screenshot should move fast.

Capture

Global hotkey, custom region overlay, and fast selection flow.

Markup

Lightweight arrows, boxes, pen, and highlights.

Copy

Clipboard-first export with disposable scratch files.

Download

Durable PNG saves only when explicitly requested.

Local First

No cloud sync, account system, or telemetry.

Not A Full Editor

It is meant to stay quick, not become a design suite.

Public Baseline

This is one of the clearest small shipped Manna utilities.

Subtraction First

Future polish should stay simple and useful.

Manna Snips is proof that small, concrete software can ship quickly when the use case stays real.
