A local awareness tool for the owner's own network.

Manna-Watch is planned as a Windows dashboard for seeing devices on a trusted local network, remembering what is familiar, and flagging what is new or uncertain.

The first version should stay grounded in the owner's environment. Broader nearby-signal or public-technology awareness can wait until the local trust core is useful and lawful.

Best MVP

Scan local network, list devices, identify likely device type, remember devices, and flag unknown arrivals.

Truth rule

Say likely, possible, unknown, or familiar instead of faking certainty.

Owner Environment

Start with the user's own local network.

Not Covert

No invasive or secret collection behavior.

Desktop First

A simple Windows dashboard is the preferred first surface.

Map Later

Do not start as a map product before device trust works.

Manna-Watch is the cautious awareness lane: useful local trust before dramatic signal maps.
