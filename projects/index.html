How To Read This Map

Visibility does not equal public access

Public Route

A web page that is intentionally reachable and useful for explaining the project.

Public Product

A thing strangers can actually use. Very few Manna systems should claim this early.

Private System

A real tool that belongs on local machines, private accounts, or controlled workflows.

Saved Plan

A project spine or direction worth preserving, but not something to market as active product yet.

This project map should keep changing as the work changes. The rule is simple: make the ecosystem easier to understand without flattening private prototypes into fake public products.
