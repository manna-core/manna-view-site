A useful upgrade kit for a serious outside system.

The Jericho Upgrade Kit is a public-safe package Grayson and Manna built to help improve Jericho / Marvin / Thought Before Action style projects without exposing private internals.

It packages Manna-style project spines, safety models, memory architecture, prompts, install scripts, and upgrade workflows into something another builder can use.

Current reality

The kit exists as a public GitHub-ready package with docs, prompts, installers, examples, and safety-first upgrade flows.

Boundary

Public page should not expose client data, secrets, handoff tokens, or vulnerable implementation details.

Project Spine

Helps a target repo establish current state, next steps, roadmap, task state, and decisions.

Safety Pass

Guides review of secrets, auth boundaries, public functions, deployment risk, and nonbreaking changes.

Upgrade Batches

Encourages small verified improvements instead of risky broad rewrites.

Gift Shape

Built as something helpful and personal, not as generic consulting fluff.

Public Safe

Contains reusable workflows, not private secrets.

Nonbreaking

Improvement should respect the working system already there.

Manna Pattern

State, safety, memory, verification, and next moves travel as a kit.

The Jericho Upgrade Kit shows Manna helping outside the local workspace in a way that is useful, bounded, and safe to talk about publicly.
