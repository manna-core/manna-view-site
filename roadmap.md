# Website Roadmap

## Mission

Make `D:\Manna-core\projects\Website` the clean public source for `manna-core.dev`:

- professional at first glance
- truthful about project status
- flexible enough to host multiple project routes
- clearly related to the wider `manna` identity without flattening every route into one tone

## Current Phase

Phase 1 - Public root and route hardening

The focus right now is establishing a cleaner umbrella homepage, stronger shared UI, and clearer
copy for the routes that are already public.

## Phase Plan

### Phase 1 - Shared Public Foundation

- improve the root landing page
- improve shared styling across public routes
- tighten wording so the site reads like a real public home instead of internal notes dumped onto a page

### Phase 2 - Better Project Surfaces

- deepen the `Projects` hub
- keep adding stronger detailed profiles for the most important builds
- distinguish live routes, public briefs, and private systems more clearly

### Phase 3 - Domain-Level Expansion

- prepare for future subdomains or dedicated app routes when projects earn them
- strengthen metadata, preview cards, and external trust signals
- keep the root site acting like an umbrella instead of a catch-all junk drawer
