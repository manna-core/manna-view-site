Manna-View

Place-first workflow Lawful public signals Active independent build

A map-first system for lawful public-world intelligence.

Manna-View is being built to make public-world visibility cleaner, faster, and more spatially understandable. The core workflow starts with a place, then gathers lawful public signals, route context, and source-aware map intelligence around it.

It is not meant to be a cluttered feed wall. The intended product is a calm, map-native intelligence surface where geography, movement, and visible sources do the heavy lifting.

See the wider project network Back to manna-core.dev
Mission

Search a place, gather lawful public signals, and map them cleanly for awareness, oversight, and context.

What Is In Scope

Cameras, routes, alerts, weather, vessels, aircraft, and other map-visible public data layers that benefit from spatial context.

Boundary

Built for public transparency and public-interest understanding, not covert surveillance or evasion tooling.

Designed To Answer

Three questions quickly and clearly

What Is Here

The map should reveal relevant public signals without burying the place under base-map clutter or dashboard sprawl.

What Is Changing

Movement, route context, alerts, and local signal shifts should feel visible at a glance instead of requiring cross-tool scavenging.

How Trustworthy Is It

Sources, status, and degraded conditions should stay visible so the operator can tell the difference between live signal and partial visibility.
